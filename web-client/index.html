  1. Android Setup: Install and run the BLE Bridge Android app

  3. Browser Setup: Enable experimental features in Chrome (see warning above)

  5. Read Messages: Click "Read Message" to get the current broadcast message
  6. Send Messages: Type in the input box and click "Send" (updates Android's broadcast)

üí° How It Works:

This connects to your Android device via BLE GATT to read the messages being broadcast in advertisements. While the Android app broadcasts via advertisements, this web client connects to read and update those messages.
